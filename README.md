# dotfiles

## Vim

### Pre-requisites
1.  Set up [Pathogen](https://github.com/tpope/vim-pathogen) for easier management of Vim plugins:
    ```sh
    $ mkdir -p ~/.vim/autoload ~/.vim/bundle && \
          curl -LSso ~/.vim/autoload/pathogen.vim https://tpo.pe/pathogen.vim
    ```
